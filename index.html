⚙️ Configurar Firebase

Firebase sincroniza la lista y los check-ins entre tu laptop y celular en tiempo real. Es gratis.

  1. Ve a console.firebase.google.com
  2. Crea un proyecto → dale cualquier nombre
  3. En el menú lateral: Realtime Database → Crear base de datos (modo prueba)
  4. Ve a Configuración del proyecto ⚙️ → Tus apps → </> Web
